# NEON SURVIVOR

A minimalist canvas-based survival shooter built using HTML, CSS and Vanilla JavaScript.

## Goals
- Understand real-time game loops
- Implement frame-rate independent movement
- Build collision systems from scratch
- Practice clean project architecture

## Tech Stack
- HTML5 Canvas
- Vanilla JavaScript
- CSS

## Current Features
- Fullscreen canvas
- Frame-rate independent movement
- Vector normalization
- Boundary clamping
- Mouse tracking
